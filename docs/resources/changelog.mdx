## 0.1.0

Initial public documentation for the Archive Order API.

Includes:

- Order creation, listing, retrieval, and withdrawal.
- Offer listing, selection, and decline.
- Payment initiation, snapshot retrieval, and finalization.
- Order messages, delivered documents, and event history.
- Webhook endpoint management.
- Webhook event signature documentation.
- OpenAPI 3.1 contract.
